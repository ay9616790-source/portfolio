## 💻 Getting Started (Local Development)

To run this portfolio locally on your machine, you need to run both the Frontend (React) and the Backend (Node.js email server).

### 1. Setup the Frontend
```bash
git clone https://github.com/ay9616790-source/portfolio.git
cd portfolio
npm install
npm run dev
```

### 2. Setup the Email Backend (for Contact Form)
Open a **new terminal tab** in the same `portfolio` directory and run:
```bash
npm install express nodemailer cors
node email-server.js
```
The local API will start on `http://localhost:8080`.
